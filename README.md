# ДЕМОНСТРАЦІЯ РОЗРАХУНКУ ВАРТОСТІ МЕБЛІВ
## Furniture Price Calculation Demo

Система розрахунку вартості та технічних специфікацій для кухонних меблів.  
**Проект:** Класична кухня МДФ дуб сонома

---

## ОГЛЯД ПРОЕКТУ

Цей репозиторій містить повний комплект документації для виробництва класичної кухні:
- Технічні розрахунки матеріалів і вартості
- Робочі креслення та специфікації
- Рекомендації по фурнітурі та комплектації
- Інструкції з виробництва та збірки

### Характеристики кухні:
- **Стиль:** Класичний
- **Матеріал фасадів:** МДФ 19мм, дуб сонома
- **Корпус:** ДСП 18мм, дуб сонома
- **Розміри:** Ліва стіна 3000мм + Права стіна 4300мм
- **Фурнітура:** Soft-close (Blum, GTV, Hettich)

---

## СТРУКТУРА ФАЙЛІВ

### Основні модулі:
```
kitchen_calculator.py       - Головний калькулятор

```

### Вихідні документи:
```

cutting_list.csv                - Деталювальна відомість
cutting_maps.txt               - Карти розкрою матеріалів
hardware_specification.csv      - Специфікація фурнітури
```

### Технічна документація:
```
technical_specifications.md      - Технічні креслення та розміри
hardware_recommendations.md     - Рекомендації по фурнітурі
production_schedule.md          - Календарний план виробництва
assembly_instructions.md        - Інструкція зі збірки
```

## ТЕХНІЧНІ ХАРАКТЕРИСТИКИ

### Габарити:
- **Загальна довжина:** 7,300 мм
- **Висота нижніх тумб:** 850 мм (з стільницею)
- **Висота верхніх шаф:** 800 мм
- **Глибина нижніх тумб:** 600 мм
- **Глибина верхніх шаф:** 400 мм

### Конструкція:
- **Нижні тумби:** 8 модулів (800, 600, 600, 1000, 600, 600, 600, 600 мм)
- **Верхні шафи:** 9 модулів по 600 мм
- **Стільниця:** Суцільна, 38 мм товщина
- **Цоколь:** 100 мм висота

### Вбудована техніка:
- Варильна поверхня (600×600 мм)
- Посудомийна машина (600×570×820 мм)
- Холодильник (окремостоячий)

## РОЗШИРЕННЯ СИСТЕМИ

### Можливості модифікації:
1. **Додавання нових матеріалів:**
   ```python
   materials["new_material"] = Material(
       name="New Material",
       name_ua="Новий матеріал", 
       unit="м²",
       price_per_unit=500.0
   )
   ```

2. **Зміна конфігурації кухні:**
   - Корегування розмірів стін

3. **Оновлення цін:**

   - Зміна цін на фурнітуру
