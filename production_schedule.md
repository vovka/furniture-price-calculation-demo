# КАЛЕНДАРНИЙ ПЛАН ВИРОБНИЦТВА ТА ОЦІНКА РОБІТ
## Production Schedule and Work Estimates

**Проект:** Класична кухня МДФ дуб сонома  
**Загальна тривалість:** 12-15 робочих днів  
**Дата початку:** За домовленістю  

---

## 1. СТРУКТУРА РОБІТ ТА ТЕРМІНИ

### Фаза 1: Підготовка та планування (1-2 дні)
- [ ] Остаточне узгодження проекту з замовником
- [ ] Перевірка розмірів на об'єкті (виїзд замірника)
- [ ] Корегування креслень при необхідності
- [ ] Замовлення матеріалів та фурнітури
- [ ] Складання програми розкрою

### Фаза 2: Виробництво корпусів (4-5 днів)
- [ ] Розпил ДСП для корпусів
- [ ] Свердління присадки
- [ ] Кромкування деталей
- [ ] Контроль якості деталей корпусу
- [ ] Підготовка до збірки

### Фаза 3: Виробництво фасадів (3-4 дні)
- [ ] Розпил МДФ для фасадів
- [ ] Фрезерування профілю (класичний стиль)
- [ ] Шліфування фасадів
- [ ] Покриття декором "дуб сонома"
- [ ] Свердління під петлі
- [ ] Контроль якості фасадів

### Фаза 4: Підготовка стільниці (1 день)
- [ ] Розкрій стільниці
- [ ] Вирізка отворів під мойку та варильну поверхню
- [ ] Обробка торців
- [ ] Контроль розмірів

### Фаза 5: Збірка та комплектація (2-3 дні)
- [ ] Попередня збірка корпусів
- [ ] Встановлення фурнітури
- [ ] Регулювання петель та направляючих
- [ ] Комплектація всіх деталей
- [ ] Пакування для транспортування

## 4. ГРАФІК ПЛАТЕЖІВ

### Варіант 1: Поетапна оплата
- **Передоплата (30%):** При підписанні договору - 12,114 грн*
- **Проміжний платіж (50%):** Після виготовлення корпусів - 20,190 грн*
- **Остаточний розрахунок (20%):** Після встановлення - 8,076 грн*

*\*Розраховано від загальної суми проекту 40,379 грн*

### Варіант 2: Оплата за роботи
- **При замовленні:** Матеріали + 50% робіт - 41,996 грн
- **Після встановлення:** Залишок за роботи - 7,344 грн

## 6. УМОВИ ВИКОНАННЯ РОБІТ

### Вимоги до об'єкта:
- [ ] Завершені оздоблювальні роботи стін та стелі
- [ ] Підведена електропроводка (220V, заземлення)
- [ ] Підведені комунікації для техніки
- [ ] Рівна підлога (перепад не більше 3 мм на 1 м)
- [ ] Вільний доступ для доставки меблів

### Відповідальність сторін:
**Виконавець відповідає за:**
- Якість матеріалів та виробництва
- Дотримання термінів виконання
- Правильність встановлення
- Гарантійне обслуговування

**Замовник забезпечує:**
- Своєчасну оплату згідно графіку
- Доступ до об'єкта
- Готовність об'єкта до встановлення
- Приймання робіт

---

## 7. ГАРАНТІЙНІ ЗОБОВ'ЯЗАННЯ

### Термін гарантії:
- **Меблі (корпуси, фасади):** 24 місяці
- **Фурнітура:** 36 місяців (згідно гарантії виробника)
- **Стільниця:** 12 місяців
- **Підсвітка:** 12 місяців
- **Роботи з встановлення:** 12 місяців

### Гарантійні випадки:
- Дефекти матеріалів
- Браковані деталі
- Неправильне встановлення
- Поломка фурнітури

### Не покривається гарантією:
- Механічні пошкодження
- Неправильна експлуатація
- Дія вологи (при порушенні умов експлуатації)
- Природний знос

---

**Примітка:** Всі терміни та вартості є орієнтовними і можуть корегуватися після остаточного узгодження проекту та умов виконання робіт.
