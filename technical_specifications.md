# ТЕХНІЧНА СПЕЦИФІКАЦІЯ КУХОННИХ МЕБЛІВ

**Проект:** Класична кухня, МДФ дуб сонома

**Замовник:** Згідно ТЗ

- **Стиль:** Класичний

- **Корпус:** ДСП ламіноване 18мм, дуб сонома
- **Полиці:** ДСП ламіноване 16мм, дуб сонома
- **Стільниця:** 38мм, дуб сонома
- **Задні стінки:** ДВП 3мм
- **Кромка:** ПВХ 2мм, дуб сонома

### Габарити кухні:
- **Ліва стіна:** 3000 мм
- **Права стіна:** 4300 мм
- **Висота верхніх шаф:** 800 мм
- **Глибина верхніх шаф:** 400 мм
- **Глибина нижніх шаф:** 600 мм
- **Висота нижніх шаф з стільницею:** 850 мм
- **Висота цоколя:** 100 мм

## 3. ТЕХНІЧНІ КРЕСЛЕННЯ ТА РОЗМІРИ

### 3.1 Нижні тумби (стандартна секція 600мм)

```
Вид спереду:
┌─────────────────┐ ← 600 мм
│                 │
│     ФАСАД       │ ← 720 мм (висота без стільниці)
│                 │
└─────────────────┘

Вид збоку:
┌─────────────────┐ ← 600 мм (глибина)
│  ┌───────────┐  │
│  │           │  │ ← 597 мм (внутрішня глибина)
│  │  ПОЛИЦЯ   │  │
│  │           │  │
│  └───────────┘  │
└─────────────────┘
```

### Конструктивні деталі нижньої тумби 600мм:
- **Фасад:** 600 × 720 × 19 мм (МДФ)
- **Бокові стінки:** 597 × 720 × 18 мм (2 шт., ДСП)
- **Верх/низ:** 564 × 597 × 18 мм (2 шт., ДСП)
- **Полиця:** 564 × 597 × 16 мм (1 шт., ДСП)
- **Задня стінка:** 600 × 720 × 3 мм (ДВП)

### 3.2 Верхні шафи (стандартна секція 600мм)

```
Вид спереду:
┌─────────────────┐ ← 600 мм
│                 │
│     ФАСАД       │ ← 800 мм
│                 │
└─────────────────┘

Вид збоку:
┌─────────────┐ ← 400 мм (глибина)
│ ┌─────────┐ │
│ │ ПОЛИЦЯ  │ │ ← 397 мм (внутрішня глибина)
│ └─────────┘ │
└─────────────┘
```

### Конструктивні деталі верхньої шафи 600мм:
- **Фасад:** 600 × 800 × 19 мм (МДФ)
- **Бокові стінки:** 397 × 800 × 18 мм (2 шт., ДСП)
- **Верх/низ:** 564 × 397 × 18 мм (2 шт., ДСП)
- **Полиця:** 564 × 397 × 16 мм (1 шт., ДСП)
- **Задня стінка:** 600 × 800 × 3 мм (ДВП)

### 3.3 Стільниця
- **Розміри:** 7300 × 600 × 38 мм
- **Матеріал:** Постформінг, дуб сонома
- **Підрізка:** Під мойку, варильну поверхню

## 4. ФУРНІТУРА ТА КОМПЛЕКТАЦІЯ

### 4.1 Петлі
- **Тип:** Накладні петлі з інтегрованими доводчиками
- **Бренд:** Blum Clip Top або аналог
- **Кількість:** 26 шт. (2 петлі на дверцята)
- **Кут відкриття:** 110°

### 4.2 Направляючі для ящиків
- **Тип:** Повного висуву з доводчиками soft-close
- **Бренд:** Blum Tandem або аналог
- **Довжина:** 500 мм
- **Кількість:** 8 комплектів

### 4.3 Ручки
- **Тип:** Класичні меблеві ручки
- **Матеріал:** Металеві, під бронзу
- **Кількість:** 21 шт.
- **Міжосьова відстань:** 128 мм

### 4.4 Цоколь
- **Висота:** 100 мм
- **Матеріал:** ДСП, дуб сонома
- **Регулювання:** Регульовані опори

## 7. ПОРЯДОК ЗБІРКИ ТА ВСТАНОВЛЕННЯ

### 7.1 Підготовчі роботи
1. Перевірка розмірів приміщення
2. Підготовка стін (рівність, електропроводка)
3. Розмітка розташування шаф

### 7.2 Встановлення нижніх тумб
1. Встановлення та регулювання цоколя
2. Збірка корпусів тумб
3. Встановлення по рівню
4. Кріплення до стіни

### 7.3 Встановлення стільниці
1. Підгонка та підрізка під техніку
2. Кріплення до корпусів
3. Герметизація стиків

### 7.4 Встановлення верхніх шаф
1. Розмітка висоти (1500-1600 мм від підлоги)
2. Кріплення до стіни
3. Вирівнювання по горизонталі

### 7.5 Установка фасадів та фурнітури
1. Регулювання петель
2. Встановлення ручок
3. Підключення доводчиків

### 7.6 Підключення підсвітки
1. Прокладка кабелю
2. Встановлення LED стрічки
3. Підключення та тестування

## 8. ТЕХНІЧНІ ВИМОГИ ДО ВИРОБНИЦТВА

### 8.1 Розпил матеріалів
- **Точність:** ±1 мм
- **Якість різу:** Без сколів
- **Програма розкрою:** Оптимізована

### 8.2 Кромкування
- **Клей:** PUR (поліуретановий)
- **Температура:** 180-200°C
- **Обрізка:** Верх/низ + торці

### 8.3 Свердління
- **Присадка під петлі:** ø35 мм, глибина 13 мм
- **Присадка під полиці:** ø5 мм, з кроком 32 мм
- **Конфірмат:** ø8 мм

### 8.4 Фрезерування фасадів
- **Профіль:** Класичний (згідно ескізу)
- **Глибина:** 3-5 мм
- **Обробка:** Шліфування P240
